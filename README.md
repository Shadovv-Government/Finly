# Finly

Finly — это учебный проект, разрабатываемый в рамках кейса «Создание PWA‑приложения для управления личными финансами» (МАИ, 01.03.02 — Прикладная математика и информатика). Приложение позволяет пользователям вести учёт доходов и расходов, анализировать финансовые привычки и планировать бюджет даже без подключения к интернету.

## Функциональность

### Основные возможности

#### Операции
- Добавление доходов и расходов.
- Поля: сумма, тип, категория, дата/время, комментарий.
- Редактирование и удаление операций.

#### История операций
- Список всех операций.
- Фильтры по периоду (день / неделя / месяц / произвольный диапазон).
- Фильтры по категориям.
- Поиск по тексту комментария.

#### Категории
- Базовый набор категорий (еда, транспорт, жильё и т.д.).
- Пользовательские категории.
- Цвет и иконка для каждой категории.

#### Аналитика
- Баланс за выбранный период.
- Суммы расходов по категориям.
- Графики и визуализация данных (структура расходов, динамика по времени).

### Расширенные возможности

#### Бюджеты и лимиты
- Месячные и недельные бюджеты по категориям.
- Индикатор прогресса и предупреждения о перерасходе.

#### Цели и «копилки»
- Создание финансовых целей.
- Прогресс‑бар по каждой цели и расчёт необходимого ежемесячного вклада.

#### Повторяющиеся операции
- Шаблоны для регулярных платежей (аренда, подписки и т.п.).
- Автодобавление по расписанию.

#### Импорт/экспорт данных
- Экспорт операций в CSV/JSON.
- Импорт из CSV для переноса истории.

#### Мультивалютность
- Поддержка нескольких валют.
- Пересчёт по курсу.

#### Темы и персонализация
- Светлая и тёмная темы.
- Базовые настройки интерфейса под предпочтения пользователя.

### Киллер‑фичи (умный ассистент)

- Быстрый ввод в свободной форме. Обработка коротких голосовых и текстовых фраз с извлечением суммы, категории и комментария из естественного языка (например, «кофе 450 рублей в Старбаксе»).
- Адаптивная автокатегоризация. Модель подстраивается под пользователя: учится на его исправлениях и всё точнее относит новые операции к нужным категориям.
- Персонализированная аналитика и прогнозы. Построение краткосрочных прогнозов бюджета («при текущем темпе ты потратишь ~N ₽ до конца месяца»), поиск аномалий и нетипичных трат.
- Рекомендации по бюджету и целям. Понятные советы, как перераспределить расходы и сколько откладывать, чтобы уложиться в бюджет и достигать целей («снизив доставки еды на 20%, освободишь X ₽ в месяц»).
- AI‑обзор периода одним экраном. Краткий дайджест за неделю или месяц: где перерасход, какие категории выросли сильнее всего, какие 2–3 действия дадут максимальный эффект.
- Мини‑чат‑ассистент по своим финансам (опционально). Возможность задать вопрос в формате чата («сколько я потратил на кафе за прошлый месяц?», «из чего урезать, чтобы отложить 10 000 ₽») и получить ответ на основе собственных данных.

## PWA и Offline

- Работа без интернета: приложение открывается и работает offline, данные сохраняются локально в IndexedDB.
- Установка как приложения: добавление на домашний экран и запуск в «приложенческом» режиме.
- Поддержка мобильных и десктопных устройств (adaptive / responsive‑верстка).

## Технологии

### Frontend
- TypeScript, JavaScript, HTML5, CSS3.
- React 18+.
- React Router.
- UI‑библиотека: Material‑UI (MUI) / Chakra UI.
- Управление состоянием: Zustand / Redux Toolkit.
- Графики: Chart.js / Recharts.

### Хранение данных
- Локальное хранилище: IndexedDB.
- Библиотека: Dexie.js.

### PWA
- Сборщик: Vite.
- PWA‑плагин: vite-plugin-pwa.
- Service Worker: Workbox.
- Manifest: Web App Manifest.

### Инфраструктура
- Git: GitHub.
- Деплой: Vercel / Netlify.
- CI/CD: GitHub Actions (сборка, тесты, деплой).
